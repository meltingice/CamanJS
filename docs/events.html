events.js

¶

The event system in Caman is responsible for registering and firing events at certain points during the render process. It is fairly straightforward and consists of a listen() and trigger() function.

The listen() function takes a type (string describing what event to listen for) and a callback that is fired when the event occurs. The trigger() function takes a type (name of the event) and some optional data to pass to the registered callbacks that will be fired.
